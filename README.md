本实验在 MNIST 手写数字数据集上，对比了 BP 神经网络（MLP） 与 卷积神经网络（CNN） 的分类性能。

1. BP 神经网络（MLP）

采用多层感知机结构，输入为 28×28 的灰度图像，首先将图像展平成 784 维向量，再通过全连接层完成分类。
网络结构为：

输入层：784

隐藏层：256 → 128

输出层：10（对应数字 0–9）

训练参数设置为 Adam 优化器，学习率为 0.001，训练 15 个 epoch。
最终在测试集上的分类准确率达到 98% 以上，满足作业要求。

MLP 结构简单，但由于需要将图像展平，会丢失空间位置信息，因此分类性能存在一定上限。

2. 卷积神经网络（CNN）

CNN 直接以二维图像作为输入，通过卷积层提取局部特征，能够更好地保留图像的空间结构信息。
本实验中 CNN 网络结构为：

卷积层：

Conv1：1 → 32

Conv2：32 → 64

池化层：两次 2×2 MaxPooling

全连接层：256

输出层：10

训练参数与 MLP 保持一致，使用 Adam 优化器，学习率为 0.001，训练 15 个 epoch。
最终测试集分类准确率达到 99% 以上，明显优于 MLP。

3. 对比分析

实验结果表明，在图像分类任务中，CNN 能够利用局部感知和参数共享机制提取更有效的特征，因此在相同训练条件下，其分类准确率明显高于传统的 BP 神经网络（MLP）。
